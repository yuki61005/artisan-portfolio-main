# Artisan Portfolio — Blog Publishing Web App

A full-stack blog publishing platform inspired by Medium, built with React + TypeScript + Supabase.

![App Screenshot](blogs.png)

## Features

- 📝 Create, edit, and publish blog articles
- 🔐 User authentication (Sign up / Login)
- 🗄️ Real-time database with Supabase (PostgreSQL)
- 📱 Responsive design with Tailwind CSS
- ⚡ Fast performance with Vite

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React + TypeScript |
| Styling | Tailwind CSS |
| Backend / DB | Supabase (PostgreSQL) |
| Build Tool | Vite |
| Deploy | Cloudflare Workers |
